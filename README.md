# ShopEZ---E-Commerce-Platform
# 🛒 ShopEZ

ShopEZ is a full-stack eCommerce web application built using the **MERN stack (MongoDB, Express, React, Node.js)**.
It allows users to browse products, create accounts, and interact with a modern online shopping platform.

---
## Live Demo
https://shop-ez-e-commerce-platform.vercel.app/
## 🚀 Features

* 🔐 User authentication (Signup / Login)
* 🛍️ Browse products
* 🛒 Shopping cart functionality
* 📦 Backend API with Express
* 🗄️ MongoDB database integration
* ⚡ Responsive React frontend

---

## 🛠️ Tech Stack

**Frontend**

* React

**Backend**

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/Jayant817/shopez.git
```

Go to project folder

```
cd shopez
```

### Install backend dependencies

```
cd backend
npm install
```

### Install frontend dependencies

```
cd ../frontend
npm install
```

---

## ▶️ Run the Application

Start backend server

```
cd backend
npm run dev
```

Start frontend

```
cd frontend
npm run dev
```

---

## 🌐 Environment Variables

Create a `.env` file inside the backend folder and add:

```
MONGO_URI=your_mongodb_connection_string
PORT=5000
JWT_SECRET=your_secret_key
```
